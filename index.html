What is an NFT

NFT stands for “Non-fungible token” and is a cool way of saying it’s a truly unique digital item that YOU can buy, own, and trade.

How do I mint a Solmosquito
How to mint on a computer

If you are using a computer to mint our lovely Solmosquitoes, download and install a Chrome/Firefox browser plugin for Phantom wallet. This will allow websites (that you authorize) to access to your Sol wallet. If you have just created a new Phantom wallet account, buy some Sol from an exchange. Once you have the wallet installed, this website will recognize it and add buttons that allow you to mint Solmosquito directly on our website.

How to mint on phone

If you are using a mobile phone to mint our lovely solmosquitoes, you need to use the solflare application built-in browser to mint our NFTs. Therefore, please launch the solflare application, click the 3 lines on the top left menu in the application and select "Browser". If you have just created a new Solflare wallet account, buy some Sol from an exchange. Once you have the wallet installed, this website will recognize it and add buttons that allow you to mint Solmosquito directly on our website.
